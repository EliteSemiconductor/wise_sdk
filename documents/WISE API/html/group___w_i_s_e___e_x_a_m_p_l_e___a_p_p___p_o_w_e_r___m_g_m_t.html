Detailed Description

Example application: idle/sleep/shutdown power modes with wakeup triggers.

This demo uses the following Core API modules:

  • WISE System APIs for entering low-power modes (idle/sleep/shutdown)
  • WISE Wake-Up Timer APIs for wakeup timer configuration
  • WISE GPIO APIs for GPIO wakeup interrupt configuration
  • WISE UART APIs for shell UART transport
  • WISE Core APIs for main processing loop

Shell commands:

  • idle : enter CPU idle mode, wake by WUTMR or GPIO
  • sleep : enter system sleep mode, wake by WUTMR or GPIO
  • shutdown : enter shutdown mode, reboot after wakeup timer or GPIO trigger
