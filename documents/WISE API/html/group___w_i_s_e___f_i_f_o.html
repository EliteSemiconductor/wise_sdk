◆ WISE_FIFO_REF

typedef void* WISE_FIFO_REF

Opaque reference to a FIFO instance.

The actual FIFO structure is hidden from users. All operations are performed through this handle.

◆ wise_fifo_create()

WISE_FIFO_REF wise_fifo_create ( uint8_t * buffer,
uint16_t bufLen )

Create and initialize a FIFO on top of a user-provided buffer.

The caller supplies a buffer and its length; the FIFO implementation uses this memory for queued data. The buffer must remain valid for the lifetime of the FIFO.

Parameters
[in]bufferPointer to memory used as FIFO storage.
[in]bufLenSize of buffer in bytes.
Returns
A FIFO reference on success, or NULL on failure.
