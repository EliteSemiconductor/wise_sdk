Detailed Description

Example application: Flash control through shell commands.

This demo integrates a shell backend over UART and exposes flash management commands built on top of Core APIs:

  • WISE Flash & OTP APIs (read/write/erase/info)
  • WISE UART APIs (shell transport)
  • WISE Core APIs (main processing loop)
  • WISE System APIs (system services)

Supported shell commands:

  • read : Read flash data and dump buffer
  • write : Write pattern data to flash
  • erase : Erase flash sector
