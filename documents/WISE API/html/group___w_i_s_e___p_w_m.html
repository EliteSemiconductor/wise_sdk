◆ PWM_MODE_T

enum PWM_MODE_T

PWM operation mode.

Period mode uses explicit WISE_PWM_CONF_T::u::period.period and WISE_PWM_CONF_T::u::period.active_period. Frequency mode instead uses WISE_PWM_CONF_T::u::freq.frequency_Hz and WISE_PWM_CONF_T::u::freq.duty_percent.

Enumerator
PWM_MODE_PERIOD 

Define PWM using period and active-period (ticks).

PWM_MODE_FREQUENCY 

Define PWM using frequency and duty cycle (%).

WISE_STATUS wise_pwm_configure ( uint8_t channel,
uint8_t gpio_index,
WISE_PWM_CONF_T * pwm_conf )

Configure a PWM channel.

Binds the PWM channel to a GPIO pin and applies PWM configuration parameters.

Parameters
[in]channelPWM channel index.
[in]gpio_indexGPIO pin index to output the PWM waveform.
[in]pwm_confPointer to WISE_PWM_CONF_T configuration structure.
Return values
WISE_SUCCESSConfiguration successful.
WISE_FAILConfiguration failed (invalid parameters or HW error).

Reconfigure a PWM channel.

Modifies frequency configuration of a PWM channel after started. On an unconfigured PWM channel an error code WISE_FAIL will be returned.

Return values
WISE_SUCCESSConfiguration successful.
