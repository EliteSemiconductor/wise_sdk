Detailed Description

Example application: PWMSLOW continuous and one-shot output demo.

This demo uses the following Core API modules:

  • WISE Slow PWM APIs for PWMSLOW configuration/start and completion callback
  • WISE System APIs for LFOSC clock source configuration and calibration
  • WISE GPIO APIs for PWMSLOW output pin function mapping (platform dependent)

The demo supports two build modes selected by DEMO_PWMSLOW_MODE:

  • DEMO_PWMSLOW_MODE_CONTINUOUS
  • DEMO_PWMSLOW_MODE_ONESHOT

  3. Initialize PWMSLOW module
