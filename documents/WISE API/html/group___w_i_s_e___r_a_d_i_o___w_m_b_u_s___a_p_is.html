Detailed Description

Wireless M-Bus radio PHY control (Core API).

This module provides APIs to initialize and configure the radio PHY for Wireless M-Bus operation, control RX/TX, and manage interface lifecycle.

◆ wmbus_mode_t

enum wmbus_mode_t

Wireless M-Bus PHY mode selection.

Notes:

  • S1/S1m/S2 correspond to S-mode variants.
  • T1/T2 correspond to T-mode variants.
  • C1/C2 correspond to C-mode variants.
  • Some modes are marked as not supported yet.
Enumerator
WMBUS_MODE_S1 

S-mode, unidirectional.

WMBUS_MODE_S1M 

S-mode (S1m), unidirectional.

WMBUS_MODE_S2 

S-mode, bidirectional.

WMBUS_MODE_T1 

T-mode, unidirectional.

WMBUS_MODE_T2 

T-mode, bidirectional.

WMBUS_MODE_C1 

C-mode, unidirectional.

WMBUS_MODE_C2 

C-mode, bidirectional.

WMBUS_MODE_R2 

R-mode, bidirectional (not supported yet).

WMBUS_MODE_F2 

F-mode, bidirectional (not supported yet).

WMBUS_MODE_N 

N-mode (not supported yet).
