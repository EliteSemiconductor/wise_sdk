Detailed Description

Example application: shell-based loader with kermit update and boot.

This example uses the following Core APIs:

  • WISE UART APIs (UART shell I/O)
  • WISE System APIs (system remap / boot helper)
  • WISE GPIO APIs (deinit before remap)
  • WISE PMU APIs (clock gating before remap)

It also uses:

  • wise_shell_v2 for command registration
  • wise_kermit for receiving the application image
