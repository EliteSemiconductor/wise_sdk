◆ shell_is_break_requested()

bool shell_is_break_requested ( void )

Check if user requested to break current command (Ctrl+C).

Use this in long-running command loops:

while (running) {
do_work();
if (shell_is_break_requested()) {
PRINTF("Aborted by user\r\n");
break;
}
}

◆ shell_poll_break()

bool shell_poll_break ( void )

Poll for break request (call this in tight loops).

This function reads pending input and checks for Ctrl+C. Use when your command doesn't yield to shell_task().

Returns
true if Ctrl+C was pressed
