◆ wise_nfc_block_read_data()

void wise_nfc_block_read_data ( uint8_t block_idx,
uint8_t block_len,
uint32_t * rx_data_buff )

Read data from NFC memory block.

Reads block_len words starting from block_idx into rx_data_buff. Word size is implementation-specific (usually 32 bits).

Parameters
[in]block_idxBlock index to read from.
[in]block_lenNumber of blocks/words to read.
[out]rx_data_buffPointer to receive buffer for read data.
