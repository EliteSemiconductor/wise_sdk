◆ wise_i2c_transfer()

int32_t wise_i2c_transfer ( uint8_t intf,
const WISE_I2C_XFER_MSG_T * msgs,
size_t nmsgs,
uint32_t interval_ms )

Perform a combined I2C transfer.

Executes nmsgs messages in a single transaction sequence, optionally using repeated START conditions without releasing the bus in between.

Parameters
[in]intfI2C controller/channel index.
[in]msgsPointer to an array of WISE_I2C_XFER_MSG_T.
[in]nmsgsNumber of messages in the array.
[in]interval_msInterval between messages in milliseconds.
Return values
0Transfer completed successfully.
<0Transfer failed.
