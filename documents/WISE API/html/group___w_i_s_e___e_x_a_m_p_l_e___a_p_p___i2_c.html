Detailed Description

Example application: WISE I2C master/slave transfer demo.

This demo uses the following Core API modules:

  • WISE I2C APIs for I2C initialization/configuration and data transfer
  • WISE GPIO APIs for PIO pin function configuration
  • WISE System APIs for disabling SWD/TCXO when shared pins are used

The demo supports two build roles selected by DEMO_I2C_ROLE:

  • DEMO_I2C_ROLE_MASTER
  • DEMO_I2C_ROLE_SLAVE
