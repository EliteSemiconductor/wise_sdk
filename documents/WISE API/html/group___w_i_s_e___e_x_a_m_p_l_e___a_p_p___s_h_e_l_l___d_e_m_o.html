Detailed Description

Example application: UART shell using WISE Core APIs.

This demo utilizes the following Core API modules:

  • WISE UART APIs for shell I/O backend
  • WISE Core APIs for system main processing loop
  • WISE System APIs for system-level support (via demo init)

It demonstrates how to bind a shell framework to UART transport and run a continuous command-processing loop.
