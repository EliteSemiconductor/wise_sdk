Detailed Description

Example application: WUTMR control and power-mode (idle/sleep) measurement CLI.

This demo uses the following Core API modules:

  • WISE Wake-Up Timer APIs for wakeup timer counter/interrupt/start/stop
  • WISE System APIs for power management (idle/sleep)
  • WISE UART APIs for shell UART transport
  • WISE Core APIs for main processing loop

Main entry of the WUTMR CLI demo application.

Flow:

  1. Initialize demo environment and shell
  2. Register WUTMR callback via wise_wutmr_register_callback()
  3. Print banner and wait for user commands
  4. Run wise_main_proc() in the main loop
