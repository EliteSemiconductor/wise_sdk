Detailed Description

Example application demonstrating WISE_PWMSLOW low-frequency PWM output.

This demo demonstrates basic PWMSLOW output using the WISE PWMSLOW driver (Core API WISE Slow PWM APIs) with two build-time modes:

  • Continuous mode: generate a low-frequency PWM signal continuously.
  • One-shot mode : generate N pulses then stop and report completion via callback.

PWMSLOW is designed for low-frequency waveforms and uses LFOSC as its clock source, making it suitable for signals from sub-Hz up to a few Hz (platform dependent).

Build-time mode selection:

  • Define DEMO_PWMSLOW_MODE to one of:
    • DEMO_PWMSLOW_MODE_CONTINUOUS
    • DEMO_PWMSLOW_MODE_ONESHOT

Hardware notes:

  • Connect an oscilloscope (or LED) to the PWMSLOW output pin (PWMSLOW_GPIO_PIN).
  • Update pin mapping in the configuration section if needed.

Functional notes:

  • This demo configures PWMSLOW in PWMSLOW_MODE_FREQUENCY mode.
  • Default frequency: 10Hz (PWMSLOW_FREQUENCY_HZ)
  • Default duty cycle: 50% (PWMSLOW_DUTY_PERCENT)
  • One-shot mode generates PWMSLOW_ONESHOT_COUNT pulses then stops.
Note
LFOSC clock configuration and calibration are required before using PWMSLOW.
