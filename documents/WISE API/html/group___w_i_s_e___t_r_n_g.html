◆ wise_trng_disable()

void wise_trng_disable ( void )

Disable the TRNG module.

Turns off the TRNG hardware. Call this when random number generation is no longer required to reduce power consumption.

◆ wise_trng_enable()

void wise_trng_enable ( void )

Enable the TRNG module.

Activates the TRNG hardware so that random numbers can be generated. Must be called after wise_trng_init().
