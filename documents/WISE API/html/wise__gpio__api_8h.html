Detailed Description

General-Purpose I/O (GPIO) driver APIs.

This header provides the interface for configuring and managing General Purpose Input/Output (GPIO) pins, including mode selection, direction, pull resistor, debounce, drive strength, interrupt, PWM and debug bus.
