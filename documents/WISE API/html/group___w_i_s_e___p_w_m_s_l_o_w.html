Detailed Description

Low-frequency PWM configuration, control and callbacks.

These APIs are typically used to generate low-frequency PWM waveforms on a single GPIO pin (for example LED dimming, low-speed blinking, etc.).

◆ PWMSLOW_MODE_T

enum PWMSLOW_MODE_T

Slow PWM operation mode.

Period mode uses explicit WISE_PWMSLOW_CONF_T::u::period.period and WISE_PWMSLOW_CONF_T::u::period.active_period. Frequency mode uses WISE_PWMSLOW_CONF_T::u::freq.frequency_Hz and WISE_PWMSLOW_CONF_T::u::freq.duty_percent.

Enumerator
PWMSLOW_MODE_PERIOD 

Define PWM using period and active-period (ticks).

PWMSLOW_MODE_FREQUENCY 

Define PWM using frequency and duty cycle (%).

WISE_PWMSLOW_CONF_T * pwmslow_conf )

Configure slow PWM output on a GPIO pin.

Binds the slow PWM generator to a GPIO pin and applies configuration parameters such as mode, period/frequency and duty cycle.

Parameters
[in]gpio_indexGPIO pin index used for slow PWM output.
[in]pwmslow_confPointer to WISE_PWMSLOW_CONF_T configuration structure.
Return values
WISE_SUCCESSConfiguration successful.
WISE_FAILConfiguration failed (invalid parameters or HW error).
