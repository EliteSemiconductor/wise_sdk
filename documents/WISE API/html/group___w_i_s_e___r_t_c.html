enum WISE_RTC_CB_EVENT_T

RTC callback events.

Event types that can be reported to a registered callback.

Enumerator
WISE_RTC_EVENT_ALARM 

Alarm event.

WISE_RTC_EVENT_SECOND 

Per-second tick event.

WISE_RTC_EVENT_MINUTE 

Per-minute tick event.

WISE_RTC_EVENT_HOUR 

Per-hour tick event.

WISE_RTC_EVENT_DAY 

Per-day tick event.

WISE_RTC_MAX_EVENTS 

Total number of RTC events (sentinel).
