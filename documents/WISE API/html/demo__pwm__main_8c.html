Detailed Description

Example application demonstrating WISE_PWM continuous and one-shot PWM output.

This demo demonstrates basic PWM output using the WISE PWM driver (Core API WISE PWM APIs) with two build-time modes:

  • Continuous mode: generate PWM waveform continuously.
  • One-shot mode : generate N pulses then stop and report completion via callback.

Build-time mode selection:

  • Define DEMO_PWM_MODE to one of:
    • DEMO_PWM_MODE_CONTINUOUS
    • DEMO_PWM_MODE_ONESHOT

Hardware notes:

  • Connect an oscilloscope or LED to the PWM output pin (PWM_GPIO_PIN).
  • Update pin mapping in the configuration section if needed.

Functional notes:

  • This demo configures PWM in PWM_MODE_FREQUENCY mode.
  • Default frequency: 1kHz (PWM_FREQUENCY_HZ)
  • Default duty cycle: 50% (PWM_DUTY_PERCENT)
  • One-shot mode generates PWM_ONESHOT_COUNT pulses then stops.
