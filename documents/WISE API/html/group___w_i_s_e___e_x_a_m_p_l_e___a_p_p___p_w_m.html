Detailed Description

Example application: PWM continuous and one-shot output demo.

This demo uses the following Core API modules:

  • WISE PWM APIs for PWM configuration/start and completion callback
  • WISE GPIO APIs for PWM output pin function mapping (platform dependent)
  • WISE System APIs for system support (via demo init)

The demo supports two build modes selected by DEMO_PWM_MODE:

  • DEMO_PWM_MODE_CONTINUOUS
  • DEMO_PWM_MODE_ONESHOT
