Detailed Description

Example application: FreeRTOS tasks + queue messaging + UART shell.

This demo integrates the following Core API modules:

  • WISE UART APIs for shell UART backend
  • WISE GPIO APIs for LED heartbeat output
  • WISE System APIs for system services (via demo init)

And uses FreeRTOS primitives:

  • Task creation and scheduling
  • Queue for inter-task communication
